# Security-Labs

Hands-on computer security labs focused on foundational security concepts including
firewall configuration, vulnerability analysis, exploitation testing, and secure systems behavior.

## Overview
This repository contains a collection of hands-on security labs completed as part of
Computer Systems Security coursework and independent experimentation. The work emphasizes
understanding real-world security vulnerabilities, system behavior, and defensive controls
in Linux-based environments.

## Lab Environments
- Kali Linux
- Ubuntu Linux
- pfSense Firewall
- DVWA (Damn Vulnerable Web Application)
- Virtualized lab environments (VMware / VirtualBox)
